# Metadata Folder

This folder contains all metadata and configuration files required for models integrated into the OnSight Pathology.

## Contents

- `CPSAM_profiler.json`  
  Metadata for the single-cell phenotyping module.

- `cell_vit.json`  
  Metadata for the CellViT module.

- `lingshu.json`  
  Metadata for the Vision Language Model (VLM) integrated into OnSight Pathology.

- `mib_cellpose.json`  
  Metadata for the Ki-67 Cellpose module.

- `mib_mitosis.json`  
  Metadata for the mitosis detection module integrated into OnSight Pathology.

- `mib_yolo.json`  
  Metadata for the YOLO-based Ki-67 quantification model.

- `tumor_compact_kaiko_vit.json`  
  Metadata for the four-class brain tumor ViT classification model, including:
  - Meningioma
  - Metastasis
  - Glioma
  - Schwannoma

## Notes

These metadata files define model configurations, tile size, loading parameters, and additional model specific settings used by OnSight Pathology.
