# Mitosis Detector Module

This module contains the Mitosis Detector model used in the OnSight Pathology.

## Overview

The Mitosis Detector is designed to identify mitotic cells in pathology images and serves as a component of the OnSIght pathology pipeline. The model can be used for automated analysis and detection tasks within digital pathology workflows.

## Files

- `Model.py`  
  Contains the implementation of the mitosis detection model.
