# Model Training Folder

This folder contains the training code used for models integrated into the OnSight pathology.

## Contents

- `4_class_tumor`  
  Corresponds to the brain tumor ViT classification model for four tumor classes:
  - Meningioma
  - Metastasis
  - Glioma
  - Schwannoma

- `MIDOGpp`  
  Contains the training code for the mitosis detection model.  
  We thank the original authors for their contribution in developing this training framework.

- `ki67`  
  Contains the YOLO-based segmentation model used for the Ki-67 quantification task.

- `mag_detector`  
  Contains the ViT-based model for the magnification detection module integrated into OnSIght pathology.

## Additional Information

Detailed instructions and documentation are provided within each individual model training folder.
